# Changelog

所有版本的变更记录。

## [v0.0.5] - 2026-03-03

### 新功能
- **终端复制功能**: 新增选择模式，支持选择并复制终端内容，支持全选、清除选择
- **移动端优化**: 方向键（↑↓←→）和退格键支持长按连发，操作更流畅
- **Shift+Tab 按钮**: 新增到移动端虚拟键盘，方便代码缩进调整

### 改进
- 简化终端头部工具栏，移除滚动控制按钮（滚动功能仍可通过 tmux copy mode 使用）
- 改进会话持久化
- 修复语音输入焦点问题

### 文档
- 新增 [FAQ 文档](docs/FAQ.md)，解答常见问题
- README 中强调本项目 100% 由 AI (Claude Code) 编写

## [v0.0.4] - 2026-03-02

### 新功能
- **终端复制**: 支持选择模式复制终端内容，桌面端和移动端均可使用

### 改进
- 优化移动端体验

## [v0.0.3] - 2026-03-01

### 新功能
- **FAQ 文档**: 新增常见问题解答

### 修复
- 修复会话持久化问题
- 修复语音输入焦点问题
- 修复每个会话现在连接到独立的 tmux 会话

### 改进
- 所有平台统一使用 Ctrl+B 作为 tmux 快捷键

## [v0.0.2] - 2026-02-28

### 修复
- 修复移动端键盘关闭后视口缩放问题
- 处理多设备连接冲突
- 修复查看会话时删除会话的错误
- 会话配置对话框中工作目录设为只读
- 修复配置路径显示

### 改进
- 添加配置向导，改进配置加载
- 改进文档和 UI 可见性

## [v1.0.0] - 2026-02-27

> 注意：此版本后被重新编号为 v0.0.1

### 新功能
- **tmux copy mode**: 支持 tmux copy mode 滚动，可查看历史输出
- **终端历史缓冲**: 捕获 5000 行终端历史记录用于滚动显示
- **滚动条**: 可视化滚动条，显示当前滚动位置
- **Mac 快捷键**: 支持 ⌘+B 键盘快捷键
- **移动端滚动按钮**: 虚拟键盘中的专用滚动模式按钮

### 改进
- 跨平台编译支持（macOS、Linux、Windows）
- 内网穿透集成（Frp、Tailscale、Cloudflare Tunnel）

## [v0.0.1] - 2026-02-25

### 新功能
- 初始发布
- JWT Token 认证 + bcrypt 密码加密
- 玻璃拟态风格登录页
- 会话管理：创建、删除、查看终端会话
- WebSocket 实时流式传输终端输出
- 远程控制：发送命令到远程会话
- 语音输入支持
- 文件浏览器
- @ 符号触发文件引用功能
- 国际化：支持中英文切换
